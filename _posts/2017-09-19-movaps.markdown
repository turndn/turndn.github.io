title:  "Exception \#GP occurs during movaps exectution."
date:   2017-09-19 23:30:00
category: note
---

```c
#include <stdio.h>

void f(double v)
{
    printf("%lf\n", v);
}

int main()
{
    asm volatile("push %rax");
    f(1.0);
    asm volatile("pop %rax");
}
```

上記のようなプログラムを実行すると，`0x7ffff7a62824 <__printf+36>:    movaps XMMWORD PTR [rsp+0x50],xmm0` にて SIGSEGV が発生し，プロセスが強制終了してしまう．
これは memory operand が 16 byte aligned でなくなるので \#GP 例外が発生する[MOVAPS][1]．

2 回 push すれば 16 byte alinged になるので，特に問題なく動く．

```c
#include <stdio.h>

void f(double v)
{
    printf("%lf\n", v);
}

int main()
{
    asm volatile("push %rax");
    asm volatile("push %rax");
    f(1.0);
    asm volatile("pop %rax");
    asm volatile("pop %rax");
}
```

[1]: https://c9x.me/x86/html/file_module_x86_id_180.html
