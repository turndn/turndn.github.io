title:  "Grub setting for selecting kernel version on Debian"
date:   2017-09-15 22:00:00
category: note
---

```sh
$ cat /boot/grub/grub.cfg | grep menuentry
    menuentry 'Debian GNU/Linux, with Linux 4.12.0+' --class debian --class gnu-linux --class gnu --class os $menuentry_id_option 'gnulinux-4.12.0+-advanced-d8089735-563c-4a95-aac3-efa35b3804c9'
$ sudoedit /etc/default/grub
$ head /etc/default/grub
# If you change this file, run 'update-grub' afterwards to update
# /boot/grub/grub.cfg.
# For full documentation of the options in this file, see:
#   info -f grub -n 'Simple configuration'

GRUB_DEFAULT="Advanced options for Debian GNU/Linux>Debian GNU/Linux, with Linux 4.12.0+"

$ sudo update-grub
```

多分これで `update-grub` を実行するとエラーが出て怒られるので，それにしたがって `GRUB_DEFAULT` の値を直して再度 `update-grub` を実行すれば良い．
